## 🔹 1. Git Kurulu Mu Kontrol Et

Önce bilgisayarında `git` var mı kontrol et:

```bash
git --version
```

GitHub’da Repo: https://github.com/GulsahKadioglu/f

## 🔹 2. Local Projende Git Başlat

Proje klasörüne gir (şu an çalıştığın `C:\Users\gulsa\federated-cancer-screening`):

```bash
cd C:\Users\gulsa\federated-cancer-screening
git init
```

---

## 🔹 3. .gitignore Dosyası Oluştur

Docker logları, venv, cache dosyaları git’e eklenmemeli. Proje köküne `.gitignore` dosyası ekle:

`.gitignore` örneği:

```
# Python
__pycache__/
*.pyc
*.pyo
*.pyd
.env

# Docker
*.log
*.pid
*.seed
*.pid.lock
/data/
db_data/
redis_data/

# Node
node_modules/
.next/
dist/

# Pytest / cache
.pytest_cache/
.mypy_cache/
coverage/

# VSCode / IDE
.vscode/
.idea/
```

---

## 🔹 4. Dosyaları Stage Et ve Commit At

```bash
git add .
git commit -m "Initial commit: Federated Cancer Screening project"
```

---

## 🔹 5. GitHub Repo’ya Bağla

GitHub sana repo açarken şu URL’yi verdi:

```bash
git remote add origin https://github.com/<kullanici_adi>/<repo_adi>.git
```

Doğru bağlantıyı eklediğinden emin olmak için:

```bash
git remote -v
```

---

## 🔹 6. İlk Push

```bash
git branch -M main
git push -u origin main
```

---

## 🔹 7. Kontrol Et

GitHub reposuna gir → dosyaların orada görünmeli ✅

---

👉 Bundan sonra değişiklik yaptığında sadece şu üç adım yeterli:

```bash
git add .
git commit -m "Açıklama"
git push
```

---
public repo’da SSH key ayarlamadan direkt HTTPS ile push edebilirsin.

Ben sana özet olarak **ilk yükleme (initial push)** için komut sırasını vereyim (senin mevcut proje klasöründe çalıştıracaksın):

```bash
cd C:\Users\gulsa\federated-cancer-screening

# Git başlat (sadece ilk defa yapıyorsan)
git init

# Remote ekle (repo adresini GitHub sayfandan kopyala)
git remote add origin https://github.com/<kullanici_adi>/federated-cancer-screening.git

# Dosyaları ekle
git add .

# Commit mesajı ile kaydet
git commit -m "Initial commit: Federated Cancer Screening project"

# Branch'i main olarak ayarla
git branch -M main

# GitHub'a gönder
git push -u origin main
```

> Bundan sonra sadece:

```bash
git add .
git commit -m "Açıklama"
git push
```
yeterli olur
---

⚠️ Küçük bir hatırlatma: `.env` dosyaları veya gizli API key’ler varsa onları `.gitignore` içine eklemeyi unutma. Aksi halde public repo’da herkes görebilir.

proje için **.gitignore’un tam sürümünü** hazır hale getirebilirsin eğer senin için de mantıklısa
